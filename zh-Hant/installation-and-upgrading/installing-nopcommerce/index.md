# 安裝 nopCommerce

nopCommerce 是一個跨平台的解決方案，可以在 Windows 和 Linux 上執行。您也可以將 nopCommerce 安裝在 Microsoft Azure 上，或使用預先安裝好的 nopCommerce 版本，這是建立您的 nopCommerce 商店最簡單的方法。

請選擇下列其中一種方式：

- [安裝於本機 (僅供開發者使用)](xref:zh-Hant/installation-and-upgrading/installing-nopcommerce/installing-local)
- [安裝於 Windows](xref:zh-Hant/installation-and-upgrading/installing-nopcommerce/installing-on-windows)
- [安裝於 Microsoft Azure](xref:zh-Hant/installation-and-upgrading/installing-nopcommerce/installing-on-microsoft-azure)
- [安裝於 Linux](xref:zh-Hant/installation-and-upgrading/installing-nopcommerce/installing-on-linux)
- [預先安裝的 nopCommerce](xref:zh-Hant/installation-and-upgrading/installing-nopcommerce/pre-installed-nopcommerce)
- [由 Everleap 提供的全託管私有雲端代管服務](xref:zh-Hant/installation-and-upgrading/installing-nopcommerce/everleap-fully-managed-hosting)

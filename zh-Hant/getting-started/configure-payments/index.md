# 設定付款

**付款**是電子商務的核心。本章節將涵蓋允許接收付款的全球商店設定。

> [!NOTE]
>
> nopCommerce 與領先的付款提供者合作，並支援主流的付款閘道，您可以在 nopCommerce [市集](https://www.nopcommerce.com/en/marketplace) 中找到相關資訊。

本章說明如何設定付款方式、貨幣以及付款方式限制。

> [!TIP]
>
> 有關如何安裝付款（或其他任何）外掛的說明，請參閱 [nopCommerce 中的外掛](xref:zh-Hant/getting-started/advanced-configuration/plugins-in-nopcommerce) 章節。

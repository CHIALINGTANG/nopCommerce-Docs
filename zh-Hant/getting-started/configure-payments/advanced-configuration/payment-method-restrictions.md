# 付款方式限制

付款方式限制功能允許您限制來自特定國家的付款。

若要限制付款方式，請前往 **設定 → 付款限制**。

![paymentmethodrestrictions](_static/payment-method-restrictions/payment_method_restrictions.png)

勾選您想要停用該付款方式的國家/地區核取方塊。

點擊 **儲存**。

> [!NOTE]
>
> 如有需要，您可以選擇該限制欄位中的所有核取方塊，以對所有國家套用限制。

# 入門指南

如果您是 nopCommerce 的新手，請依照以下步驟來設定並執行您的商店：

## 1. 設定您的商店

- [設定一般商店選項](xref:zh-Hant/getting-started/advanced-configuration/your-store-information)
- [設定付款方式](xref:zh-Hant/getting-started/configure-payments/payment-methods/index)
- [設定配送選項](xref:zh-Hant/getting-started/configure-shipping/index)
- 設定使用者列表：建立 [不同的使用者群組](xref:zh-Hant/running-your-store/customer-management/customer-roles)，賦予他們 [對網站特定區域的個別存取權限](xref:zh-Hant/running-your-store/customer-management/access-control-list) 及優惠活動，以及更多功能。

## 2. 建立目錄並新增商品

目錄是顧客在瀏覽商店時經常互動的部分，因此，讓目錄能夠輕鬆瀏覽並找到所需的商品至關重要。

### 建立分類與製造商

分類與製造商將用於您的商品目錄，以進行商品篩選。請了解如何管理它們，以便讓網站導覽更加容易。

- [分類](xref:zh-Hant/running-your-store/catalog/categories)
- [製造商](xref:zh-Hant/running-your-store/catalog/manufacturers)

### 新增商品

您可以手動逐一新增商品，或從 XML 檔案匯入您的目錄。

- [商品](xref:zh-Hant/running-your-store/catalog/products/index)

### 定義商品變體

如果您的商品具有變體該怎麼辦？例如，您銷售各種顏色和尺寸的運動鞋。在 nopCommerce 中，您可以設定規格屬性和商品屬性來管理這些情況。

- [組合商品 (變體)](xref:zh-Hant/running-your-store/catalog/products/grouped-products-variants)

## 3. 改善商店的顧客體驗

現在您的目錄中已填入商品，請確保顧客在購物過程中獲得所需的一切支援。

### 商品評價

個人的購買體驗很重要。這能讓大眾對您身為賣家更加信任。請在此了解如何為您的線上商店啟用商品評價並進行管理。

- [商品評價](xref:zh-Hant/running-your-store/catalog/products/product-reviews)

### 即時交談 (LiveChat) 支援

通常您的顧客會需要立即提出問題並獲得解答。即時交談 (LiveChat) 選項是協助他們的好方法。您可以在 nopCommerce 市集中找到各種交談外掛。

- [深入了解 nopCommerce 中的外掛](xref:zh-Hant/getting-started/advanced-configuration/plugins-in-nopcommerce)

### 願望清單

願望清單對於延後購買是一個很好的選擇。您可以讓您的顧客將某些商品加入願望清單，以便日後購買或與朋友分享。這對您分析顧客行為來說，也是一個很好的選項。

- [購物車與願望清單](xref:zh-Hant/running-your-store/order-management/shopping-carts-and-wishlists)

## 4. 行銷與推廣工具

推廣您的商店對於獲取顧客忠誠度以及提高潛在顧客對品牌的認知至關重要。nopCommerce 提供了許多行銷選項：您可以提供特價商品、根據購物行為推薦商品、透過電子郵件活動推廣商店，以及更多其他功能。

- [推廣工具](xref:zh-Hant/running-your-store/promotional-tools/index)

### 折扣與特殊價格

為商品套用特殊價格可以鼓勵您的顧客儘快下單，以及（或）購買更多商品。

- [折扣](xref:zh-Hant/running-your-store/promotional-tools/discounts)
- [階梯價格](xref:zh-Hant/running-your-store/promotional-tools/tier-prices)

### 忠誠度計畫

nopCommerce 中的紅利點數功能允許顧客透過特定動作（註冊、購物）來賺取點數。

- [紅利點數](xref:zh-Hant/running-your-store/promotional-tools/reward-points)

### 交叉行銷

在 nopCommerce 中，您可以根據顧客的購物行為，向他們推薦 [額外商品](xref:zh-Hant/running-your-store/promotional-tools/cross-sells-and-related-products)。這同時也是您進行商品向上銷售（up-sell）的好機會。

### Email campaigns

建立 [Email 行銷活動](xref:zh-Hant/running-your-store/promotional-tools/email-campaigns)，以便輕鬆快速地向您的目標受眾傳達特別優惠、新聞、教育內容等資訊。

## 5. 適用於各類商店的 nopCommerce 特色功能

無論您是想嘗試小型專案，還是準備啟動一個電子商務市集，您都可以信賴 nopCommerce。這裡有許多選項可滿足您的業務需求：

### 多重商店 (Multi-store)

nopCommerce 讓您能透過單一的 nopCommerce 安裝與介面，執行多個商店，並從單一管理後台處理所有的管理作業。[在此了解](xref:zh-Hant/getting-started/advanced-configuration/multi-store) 如何設定多個商店。

- [多重商店](xref:zh-Hant/getting-started/advanced-configuration/multi-store)

### 多供應商

多供應商與無庫存銷售（drop shipping）工具能讓您在線上銷售商品，而無需自行維持庫存或處理訂單出貨。請觀看關於多供應商支援的 [影片教學](https://www.youtube.com/watch?v=MH6r6tqfLF8&index=9&list=PLnL_aDfmRHwsbhj621A-RFb1KnzeFxYz4)。

- [供應商管理](xref:zh-Hant/running-your-store/vendor-management)

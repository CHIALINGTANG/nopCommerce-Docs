# 設定出貨

本節將介紹在 nopCommerce 中設定出貨以及管理多個倉庫的流程。內容包含手動設定出貨方式，以及透過各別出貨提供者（如 UPS）的 API 來設定出貨。

請注意，nopCommerce 提供與第三方出貨服務的整合與最佳化功能。您可以在 nopCommerce [市集](http://www.nopcommerce.com/marketplace) 以及 [技術合作夥伴](http://www.nopcommerce.com/technology-partners) 專區找到相關資源。

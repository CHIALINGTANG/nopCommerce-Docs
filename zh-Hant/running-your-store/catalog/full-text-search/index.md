# 為您的 nopCommerce 商店設定全文檢索

全文檢索是一項強大的技術，能讓使用者快速且精確地搜尋大量文字資料。不同於 nopCommerce 預設的搜尋功能（依賴精確比對），全文檢索會分析網站內容以擷取最相關的結果。自動完成與容錯拼字功能可最佳化搜尋速度，並協助使用者找到他們真正需要的資訊。

nopCommerce 的快速全文檢索功能是透過兩個外掛來實作的：

* [基於 Lucene 的快速全文檢索](xref:zh-Hant/running-your-store/catalog/catalog-settings/full-text-search/lucene-configure)
* [基於 Elasticsearch 的快速全文檢索](xref:zh-Hant/running-your-store/catalog/catalog-settings/full-text-search/elasticsearch-configure)

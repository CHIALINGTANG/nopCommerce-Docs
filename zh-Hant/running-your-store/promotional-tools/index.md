# 推廣工具

nopCommerce 允許您使用多種推廣工具來行銷您的商店。這些工具可以協助您提高顧客對品牌的認識、鼓勵合作夥伴與供應商參與聯盟計畫、為顧客提供優惠（如折扣、階梯價格）、創造銷售業績，並建立顧客對商店的忠誠度。

# Brevo 整合

## 什麼是 Brevo

[Brevo](https://get.brevo.com/5f897rrc6cwa) 是一套全方位的數位行銷工具箱，能夠滿足您所有的行銷需求。Brevo 讓您能夠透過電子郵件行銷活動、簡訊以及精密的行銷工作流程自動化，來建立並經營顧客關係。

nopCommerce 的 Brevo 整合功能讓商店擁有者可以：

* 設計清晰且專業的電子報
* 建立並鎖定目標受眾
* 自動化行銷作業（例如：找回遺棄購物車）

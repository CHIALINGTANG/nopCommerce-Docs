# 參與貢獻的可能方式

## 以下是您可以參與 nopCommerce 專案的幾種方式

* 如果您有經營網站，請撰寫關於我們的文章或放置超連結。
* 在我們的 [論壇](http://www.nopcommerce.com/boards/) 上回答問題。
* 協助我們驗證已提交的錯誤修復。
* 在 [論壇](http://www.nopcommerce.com/boards/) 上分享關於新功能或改進現有作法的建議。
* 建立新的語言套件。
* 建立新的圖形化佈景主題並與社群分享。
* 實作新的外掛（付款閘道模組等）。
* 向您的朋友推薦 nopCommerce。

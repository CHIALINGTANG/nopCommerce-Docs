# 參與貢獻

我們非常歡迎您參與 nopCommerce 的成功，無論是協助專案的實作、參與論壇討論，或是單純地推廣分享！如果您擁有網站，歡迎撰寫關於我們的文章或放置超連結！我們致力於培育 nopCommerce 社群的成長！

nopCommerce 擁有一個年輕但不斷壯大的貢獻者社群。這是一個非常活躍的社群，熱衷於開發功能強大且基於 .NET Core 的開源電子商務應用程式。nopCommerce 在上個月的下載量超過 30,000 次，相當受歡迎。因此，加入這個社群，看看您能學習到什麼，以及能分享什麼！

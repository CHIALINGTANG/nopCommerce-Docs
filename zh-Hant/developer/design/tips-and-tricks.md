# 提示與技巧

1. 網頁開發與設計中的主要問題之一，是不同的瀏覽器對各種 CSS 屬性使用不同的預設值。在自訂或建立您的佈景主題時，請務必確保您的樣式與所有瀏覽器相容。請在 Edge、Firefox、Opera、Safari 和 Chrome 等主流瀏覽器中測試您的網站。
1. 在使用 CSS 樣式表工作時，如果您在網站上沒有看到任何變化，請不要驚慌。嘗試刪除瀏覽器的歷史記錄與 cookie，然後重新整理頁面再試一次。
1. 請務必了解 id 與 class 之間的差異。您可以將 class 套用到多個元素，而 id 僅能指向單一元素。
1. CSS 樣式表不允許您以數字或其他非字母字元作為 id 與 class 名稱的開頭。
1. 不要將自己侷限於 span 與 div 標籤。CSS 還提供了許多其他更好用的功能來設計頁面樣式。

> [!TIP]
> 請善用 Mozilla 提供的 **Firefox Developer Tools** 外掛。這是一個非常有用的工具，可讓您在任何網頁上即時編輯、除錯並監控 CSS、HTML 和 JavaScript。您可以在此處了解更多資訊：[Firefox Developer Tools.](https://developer.mozilla.org/en-US/docs/Tools)
>
> 或者，您也可以使用整合在 Chrome 瀏覽器中的 **Chrome DevTools** 工具。您可以在此處了解更多資訊：[Chrome DevTools.](https://developers.google.com/web/tools/chrome-devtools)

    響應式設計

    理解響應式設計。 響應式網頁設計（Responsive Web Design）簡而言之，就是一種會根據不同螢幕尺寸調整佈局的網站設計，目的是讓內容在桌上型電腦、智慧型手機、平板電腦及其他行動裝置上都能妥善顯示。在這個定義中，關鍵字是「妥善」（properly）—— 在此語境下，「妥善」意味著「一致性」。在進行響應式設計時，網頁開發人員必須確保網站的使用者體驗在所有裝置上都能保持一致，無論其螢幕尺寸為何。（來源：nopCommerce 部落格)

    • 行動裝置的數量正以極快的速度增加，由於螢幕尺寸與解析度各異，網頁開發人員採用響應式設計方法，以確保網站在各裝置間的使用者體驗相同或相近。

    • 是的，nopCommerce 預設的佈局與佈景主題確實支援針對所有螢幕尺寸最佳化的響應式設計，此方法是透過 CSS 中的媒體查詢（media-queries）來處理。這能讓您的電子商務商店佈局根據螢幕尺寸自動調整顯示區域，從而呈現網站內容。

    檢視響應式設計。 只需重新調整您的瀏覽器視窗大小並持續縮小。隨著瀏覽器視窗大小減小，您會看到 nopCommerce 網站會相應地調整內容。
